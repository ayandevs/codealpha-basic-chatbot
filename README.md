# 🤖 Basic Chatbot (Jarvis)

## 📖 Description

This is a simple rule-based Basic Chatbot developed in Python as part of the CodeAlpha Python Programming Internship. The chatbot interacts with the user by responding to predefined questions such as greetings, asking about its name, requesting help, and saying goodbye. It continues the conversation until the user types **"bye"**.

## ✨ Features

- Responds to basic greetings.<br>
- Answers predefined questions.<br>
- Displays a menu of available questions.<br>
- Handles unknown inputs with a default response.<br>
- Runs continuously until the user exits by typing **"bye"**.<br>
- Simple and beginner-friendly console interface.

## ▶️ How to Run

1. Install Python 3 on your computer.<br>
2. Open the terminal or command prompt.<br>
3. Navigate to the project folder.<br>
4. Run the program using:

```bash
python basic-chatbot.py
```
